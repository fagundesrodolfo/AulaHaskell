# AulaHaskell
Sistema de gestão escolar
https://ide.c9.io/romefeller/yesodaula


Usuarios: Aluno, Professor, Funcionario

Ações do Aluno:
  Fazer login,
  Ver Notas,
  Ver faltas
  
Ações Professor:
  Fazer login,
  Ver lista de Alunos por Turma,
  Inserir Faltas,
  Inserir notas
  
Ações Funcionario:
  Fazer login,
  cadastrar aluno,
  cadastrar professor,
  criar turma,
  inserir aluno em turma,
  atribuir turma a professor
